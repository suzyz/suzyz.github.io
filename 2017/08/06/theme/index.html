Problem Summary

Given an array of N integers, find the maximal length of its “repeating” subsequence.

Solution

This is a pretty staight forward dynamic programming problem.

Let diff[i]=seq[i]-seq[i-1],1<i<=n
Let f[i][j] be the maximal length of repeating subsequence ending at i and j in Diff. Then the answer is max{f[i][j]}+1.
