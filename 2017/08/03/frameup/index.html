USACO Section4.4 frameup

Problem Summary

Given a picture of several overlapping frames, find all possible sequence of the frames in the order they were stacked from bottom to top.

Solution

Note that,

It is possible to see at least one part of each of the four sides of a frame. A corner is part of two sides.

So it is easy to get a partial order of the frames. Then we only need to search for all the possible total orders.
